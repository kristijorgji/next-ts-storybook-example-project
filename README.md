## Demo project
It includes
- next.js
- typescript
- storybook
- int18
- sass support
- styled components


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Run storybook by executing in terminal:

```bash
yarn storybook
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
